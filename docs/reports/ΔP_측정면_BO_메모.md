# ΔP 측정면 — BO 단계 대비 메모 (plan_1.4V §5)

**현재:** 빈 덕트 검증에서는 측정면이 `z_in = buf_cells + 5`, `z_out = NZ - 1 - buf_cells - 5` 로 발달 구간 밖에 있어 문제없음.

**BO 시:** 도메인이 "버퍼 + 촉매층 + 버퍼" 구조이면, ΔP 측정면을 **촉매층 기준**으로 재정의해야 함.

- **Z_in** = 촉매층 시작 + 5 cells  
- **Z_out** = 촉매층 끝 − 5 cells  

이는 BO 파이프라인 구축 시 반영. 지금은 메모만.
