## 5. L2가 이론과 다른 가능 원인 (정성)

### 5.1 L2-A: ΔP_sim(0.032 Pa) ≪ ΔP_theory(0.153 Pa)

- **측정 위치:** ΔP_sim은 z_in, z_out 단면의 유체 평균 ρ 차이×p_scale. 입·출구 근처는 fully developed가 아니라 **entrance/exit 영역**이라 연속체 이론(완전 발달 흐름)과 다를 수 있음.
- **형상 해석:** 이론의 f·Re=14.227은 단일 정사각 덕트 등 특정 해석 기준. **6×6 채널 번들**에서는 채널 간 간섭·코너 효과로 유효 마찰이 달라질 수 있음.
- **LBM 특성:** MRT·Zou–He 경계조건에서 유동이 이론보다 “쉽게” 흐르면(유효 점성/저항 감소) ΔP_sim이 이론보다 낮게 나올 수 있음.
- **수치:** 같은 u_in인데 ΔP_sim이 약 1/5 수준 → 시뮬 상 유동 저항이 이론보다 **작게** 나옴 (L2-B와 반대 방향).

### 5.2 L2-B: u_channel(0.0761) ≪ u_theory(0.449)

- **동일 g(ΔP)인데 유량만 낮음** → 시뮬에서 **저항이 이론보다 훨씬 크게** 나옴.
- **형상/격자:** 6×6 이산화에서 유효 수력직경·벽 전단이 이론 가정과 다르거나, **f·Re_eff > 14.227** 로 해석될 수 있음.
- **주기 BC + 체적력:** g = ΔP/(ρ L)로 넣었으므로 압력 구배는 3.819 Pa/0.11 m로 고정. 그때 이론이면 Q ∝ ΔP이므로 u=0.449가 되어야 하는데, 시뮬은 u가 작음 → **동일 ΔP 대비 유량이 적음** = 저항 과대 평가.

### 5.3 L2-A와 L2-B 방향 차이

- **L2-A:** 같은 u_in → ΔP_sim이 이론보다 **낮음** (저항 과소).
- **L2-B:** 같은 ΔP → u_sim이 이론보다 **낮음** (저항 과대).
- 두 결과는 서로 반대 방향이므로, **경계조건·측정 정의·형상 해석**이 방법 A(입출력 압력차)와 방법 B(주기+체적력)에서 다르게 작용하는지 점검할 필요가 있음 (예: L2-A의 z_in/z_out 위치, L2-B의 u_superficial/채널 단면 정의, 6×6 공식 f·Re 사용 적합성).

### 5.4 권장 후속

- L2-A: z_in/z_out을 더 안쪽으로 옮기거나, 수렴 구간에서 ΔP 구배를 구간별로 비교.
- L2-B: 6×6에 대한 문헌/해석적 f·Re 또는 유효 Dh 재검토.
- 두 방법에서 **K_sim(A) vs K_sim(B)** 일치도 확인 (현재 K_sim(A)=1.08e-6, K_sim(B)=3.62e-8 → 스케일 차이 큼).
