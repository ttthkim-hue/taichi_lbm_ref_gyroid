## 5. STEP 3: Reference 채널

### 5.2 실행

- **ref_dx02 (dx=0.2mm)**: **완료**. step **28,900**에서 수렴(res<1e-9)으로 조기 종료. 소요 1355 s (약 0.38 h). 출력 `Results_Ref_dx02/`.
- **ref_dx04 (dx=0.4mm)**: **미실행**. 지시서 명령대로 40,000 steps 실행 시 격자 수렴 판정 가능.

```bash
# dx=0.4mm (ref_dx04)
python3 taichi_lbm_solver_v3.py \
  --mask /mnt/h/openlb_gyroid/results/gate_a/ref_dx04.npy \
  --nu 3.52e-5 --rho 0.746 --u_in 0.2778 --dx 0.0004 --tau 0.595 --steps 40000 \
  --L_cat 0.100 --out_dir /mnt/h/taichi_lbm_ref_gyroid/Results_Ref_dx04
```

### 5.3 이론값 (비교용)

- K_eff ≈ 2.31e-7 m², ΔP ≈ 2.21 Pa, Re ≈ 24.2 (a=0.003067 m, L=0.100 m, ε=0.524).

### 5.4 보고 형식 (STEP 3) — 결과

- Darcy: K = u_avg × μ × L / ΔP. μ = ν×ρ = 2.62e-5 Pa·s, L = 0.1 m, ΔP = 0.0768 Pa, u_avg ≈ 0.003 m/s (CSV) → **K(dx=0.2) ≈ 1.02e-7 m²**.
- 이론 K_th = 2.31e-7 m² → 상대 오차 약 **56%** (>5% → ❌).

```
=== STEP 3: Reference V2 ===

  K(dx=0.4): 미실행
  K 변화율: 산출 불가 (dx=0.4 미실행)
```
