### 1.2 핵심 수식 (격자–물리)

| 항목 | 식 |
|------|-----|
| 시간 간격 (plan_1.9V) | **dt = ν_lb × dx² / ν_phys**, ν_lb = (τ−0.5)/3 |
| 격자 유속 | u_lb_in = u_in_phys × dt / dx |
| 마하 | Ma = u_lb_in × √3, **Ma < 0.3** (미만족 시 예외) |
| 압력 스케일 | p_scale = ρ_phys × (1/3)(dx/dt)² → **ΔP_phys = Δρ_lbm × p_scale** |
| 주기BC 체적력 | g_phys = ΔP/(ρL), **g_lbm = g_phys × dt²/dx** (Guo forcing) |

### 1.3 경계 모드

| mode | Z=0 / Z=NZ−1 | 구동 |
|------|--------------|------|
| `velocity_inlet` (기본) | inlet 속도 BC, outlet zero-gradient+역류 clip | u_in_phys |
| `periodic_body_force` | Z 주기 스트리밍, inlet/outlet 미적용 | `set_body_force(ΔP, L)` |

## 6. 종합 결론

1. **솔버:** D3Q19 MRT + dt τ기반 고정으로 **p_scale 고정**, 저유속에서 **Δρ_lbm < 1%** 달성 가능. **주기BC+체적력**으로 고 ΔP·고 GHSV Gyroid BO 경로 확보.
2. **L1:** 물리·수치 일치 양호 (**PASS**).
3. **L2 원유속:** LBM 비압축 한계 + 127 마스크 CV → **FAIL**은 물리적으로 예측 가능.
4. **L2 131:** 균등 분할로 CV 제거; **dt 고정**으로 저유속 ΔP 검증 경로 확보. 수렴·주기BC·K 교차로 **솔버 신뢰도 2중·3중 검증** 가능.
5. **Gyroid:** ε·S_v 실측 완료; BO 전 3-GHSV K 일정성·이론 정성 비교 권장.
