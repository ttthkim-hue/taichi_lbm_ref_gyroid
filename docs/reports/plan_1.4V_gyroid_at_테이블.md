## BO 설계 변수 범위 재조정

- **t ≤ 0:** 현재 구현 `solid = 1 if |φ| < t` 에서 t=0이면 Gyroid 고체 없음(ε=1). t ∈ [−0.5, 0.5] 중 **t ≤ 0 구간은 무의미** → 실효 범위 **t ∈ (0, 0.5]** 권장.
- **min_wall_voxels:** Coarse 격자(0.4 mm)에서 위 조합은 모두 min_wall=1. **벽 두께 ≥ 3 voxel** 요구를 만족하려면 **dx 축소(Medium/Fine)** 또는 **t 상한 조정** 필요. BO 시 격자독립성(GCI) 확보 후 min_wall 재측정 권장.
- **ε 범위:** 계획서 [0.35, 0.65] 목표에 맞추려면 t를 더 크게(예: t ∈ [0.35, 0.5]) 또는 격자/해상도에 따라 재검토.

**산출물:** 위 테이블을 BO 탐색 범위 확정에 사용.
