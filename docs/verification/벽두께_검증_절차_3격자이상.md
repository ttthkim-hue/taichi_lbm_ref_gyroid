# 벽 두께 ≥3 격자 검증 절차 (§5.1·§9 R2)

**기준:** `docs/[mainplan]_V1.1.md` §5.1, §9 (R2)  
**작성 일자:** 2026-03-17

---

## 1. 요구사항

- dx = 0.2 mm 기준 **최소 3 voxel (0.6 mm)** 이상 권장. 1 mm 벽 = 5 voxel.
- **(a,t)별** 벽 두께 ≥ 3 격자 **사전 검증** (§9 R2: 형상 격자 표현 시 누수 방지).

---

## 2. 검증 절차

| 단계 | 내용 |
|------|------|
| 1 | 설계 변수 (a, t) 조합별로 형상 생성 (Taichi 커널 또는 .npy 마스크). |
| 2 | 격자 상에서 유체-고체 경계로부터 **벽 두께**(격자 수) 산출: 최소 두께 min_wall_voxel. |
| 3 | **min_wall_voxel ≥ 3** 확인. 미달 시 (a,t) 범위 조정 또는 dx 조정 검토. |
| 4 | BO 샘플 (a,t)에 대해 사전 검증 통과한 구간만 사용하거나, 런타임 검사 추가. |

---

**검증:** §5.1·§9 R2 반영 완료.
