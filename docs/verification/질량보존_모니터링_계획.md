# V-mass: 질량 보존·유량 밸런스 모니터링 계획

**기준:** `docs/[mainplan]_V1.1.md` §5.1, §10  
**작성 일자:** 2026-03-17

---

## 1. 요구사항 (§5.1)

- 매 **N스텝** 입출구 유량·총 질량 드리프트 모니터링.
- **목표:** 드리프트 < **0.1%**.
- 형상 관련: STL watertight·형상 누수 검사 필수 (STL→Voxel 미채택이어도 형상 정합성 검사 적용).

---

## 2. 모니터링 계획

| 항목 | 내용 |
|------|------|
| 주기 | 매 N 스텝 (예: 1000 또는 5000) 로그 |
| 양 | 입구 유량 Q_in, 출구 유량 Q_out, (선택) 도메인 총 질량 |
| 지표 | \|Q_in - Q_out\| / Q_in × 100 [%], 질량 드리프트 [%] |
| 기준 | < 0.1% 유지 |
