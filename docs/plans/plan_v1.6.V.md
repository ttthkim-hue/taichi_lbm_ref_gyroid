## 2. L2 Reference 6×6 본검증 실행

### 2.1 실행 전 스크립트 점검

`scripts/run_l2_ref6x6.py`에서 아래 3항목 확인:

| 항목 | 올바른 값 | 확인 포인트 |
|------|-----------|-------------|
| Inlet BC 유속 | u_in = **0.2778 m/s** (덕트 표면유속) | 채널 유속 0.449가 아님 |
| 이론 ΔP의 유속 | u_channel = **0.449 m/s** | 연속 방정식으로 산출 |
| 이론 ΔP의 길이 | L_measure = **(z_out - z_in) × dx** | 덕트 전체 길이 아님 |

### 2.2 이론값 (HP, Fanning, 정사각 단면)

6×6 채널은 완전 발달이 보장(Le=6mm ≪ 110mm)되므로 **HP 직접 비교** 가능.

```
Dh = 3.067 mm
u_channel = u_in × (A_duct / A_fluid) = 0.2778 × (548 / 339) = 0.449 m/s
Re = 0.449 × 3.067e-3 / 3.52e-5 = 39.1
f_Fanning × Re = 14.227 (정사각 단면)
f_Fanning = 14.227 / 39.1 = 0.3639

L_measure = (z_out - z_in) × dx   ← 스크립트에서 산출

ΔP = 4 × f_Fanning × (L_measure / Dh) × 0.5 × ρ × u_channel²
```

> ⚠️ 위 수치는 수기 계산. 에이전트가 직접 재계산하여 확인할 것.
> ⚠️ u_channel 사용 여부: 이론 ΔP에는 u_channel, 시뮬 inlet BC에는 u_in. 혼동 주의.

### 2.3 실행

```bash
python scripts/run_l2_ref6x6.py 2>&1 | tee run_l2_log.txt
```

수렴까지 실행. 예상 시간: ~70분.

### 2.4 PASS 기준

| 기준 | 조건 |
|------|------|
| ΔP 오차 | < **5%** (6×6은 완전발달이므로 엄격 기준 적용) |
| Q_in vs Q_out | < 1% |
| clips (정상상태) | 0 |
| CV (36채널 유량 편차) | < 10% |

### 2.5 CV 검증

출구 단면(z_out)에서 36개 채널 영역별 유량(Σ ρ·vz) 산출 → 각 채널 유량 vs Q_total/36 편차.  
대칭 구조이므로 모든 채널이 거의 동일해야 함. 큰 편차 시 마스크 오류.

### 2.6 S_v 편차 정량화

L2 스크립트 종료 전 아래 산출 후 출력:

```
S_v_theory = 36 × (4 × 3.067mm) × L_cat / (23.4mm × 23.4mm × L_cat)
           = 36 × 12.27 / 547.6  [mm/mm³ → m²/m³ 변환]

S_v_voxel = voxel 유체-고체 경계면 수 × dx² / 도메인 체적
```

편차 기록 (PASS/FAIL 대상 아님, Gyroid 보정용).
