## 1. L1 빈 덕트 — 이론값을 발달유동 상관식으로 교체

### 1.1 배경

빈 덕트 Dh = 23.4mm, Re = 184.7 → 발달 길이 Le = 216mm. 도메인 110mm에서 완전 발달 불가. 시뮬 ΔP(~0.070 Pa) vs HP 이론(~0.038 Pa) 차이 83%는 솔버 오류가 아니라 **입구 효과에 의한 정상적 차이**.

### 1.2 Shah & London 발달유동 상관식

**출처:** Shah & London (1978), 정사각 단면 (aspect ratio = 1)

```
L⁺ = L_measure / (Dh × Re)

f_app·Re = 3.44/√(L⁺) + (56.91 + 1.25/(4·L⁺) - 3.44/√(L⁺)) / (1 + 0.00021/(L⁺)²)

ΔP_theory = (f_app·Re / Re) × (L_measure / Dh) × 0.5 × ρ × u_in²
```

> ⚠️ 에이전트는 위 수식을 Shah & London 원문 또는 Incropera 교과서에서 교차 확인할 것. 수식이 틀릴 수 있음.

### 1.3 수정 대상

`scripts/run_l1_empty_duct_plan13v.py` 내 이론 ΔP 산출 부분만 교체.

**핵심 주의:**
- `L_measure` = (Z_out - Z_in) × dx. 덕트 전체 길이(110mm)가 아님. 측정면 간 실거리.
- 기존 HP 이론값은 주석 처리하고 비교 참고용으로 남겨둘 것.

### 1.4 출력 형식

```
[L1 검증 — 발달유동 상관식]
  Dh = 23.40 mm, Re = 184.7
  L_measure = {xxx} mm, L⁺ = {xxx}
  f_app·Re = {xxx}  (완전발달 극한: 56.91)
  ΔP_theory (발달유동) = {xxx} Pa
  ΔP_sim = {xxx} Pa
  오차 = {xxx}%
  판정 = PASS / FAIL (기준: 오차 < 5%)
```

### 1.5 PASS 기준

| 항목 | 기준 |
|------|------|
| ΔP 오차 | < 5% |
| Q_in vs Q_out | < 1% |
| 총 질량 드리프트 | < 0.1% |
| clips (정상상태) | 0 |

## 2. L2 Reference 6×6 — 완전 발달 유동 검증

### 2.1 배경 및 목적

6×6 채널은 발달 길이가 6mm이므로 110mm 도메인에서 **완전 발달 유동이 보장**됨. 따라서 Hagen-Poiseuille 해석해와 직접 비교 가능. L1이 발달유동 상관식을 쓸 수밖에 없었던 반면, L2는 교과서적인 HP 검증이 성립하는 케이스.

### 2.2 6×6 채널 기하 사양

| 항목 | 값 | 산출 근거 |
|------|-----|-----------|
| 덕트 내부 | 23.4 × 23.4 mm | 25.4mm − 벽 1mm × 양쪽 |
| 내부 격벽 | 5개 × 1mm (각 방향) | 6채널 사이 5개 벽 |
| 채널 폭 (a_ch) | 3.067 mm | (23.4 − 5) / 6 |
| Dh | 3.067 mm | 정사각 채널 |
| 채널 수 | 36 | 6 × 6 |
| 유로 면적 | 338.7 mm² | 36 × 3.067² |
| 채널 유속 | 0.449 m/s | u_in × (548/339) |
| Re_channel | 39.1 | 0.449 × 3.067e-3 / 3.52e-5 |
| **Le** | **6.0 mm** | 0.05 × 39.1 × 3.067 |

> Le = 6mm ≪ 110mm → 완전 발달 조건 성립.

### 2.3 이론 ΔP (HP, 정사각 단면)

```
f·Re = 56.91 (정사각 단면)
f = 56.91 / 39.1 = 1.4555
L_measure = (Z_out - Z_in) × dx  ← 측정면 간 거리 사용
ΔP_theory = f × (L_measure / Dh) × 0.5 × ρ × u_channel²
```

> ⚠️ u_channel은 u_in이 아님. 채널 유속 0.449 m/s 사용.
> ⚠️ 위 수치는 수기 계산. 에이전트가 직접 재계산하여 확인할 것.

### 2.4 Voxel 마스크 생성

6×6 Reference용 voxel 마스크를 생성하는 함수 필요.

**구현 방법 2가지 중 택 1:**
- (A) Taichi 커널로 직접 생성 — 채널/벽 판별 조건문
- (B) numpy로 마스크 생성 후 `set_geometry_from_voxel()` — 기존 인터페이스 활용

어느 쪽이든 아래 조건 충족:
- 덕트 외벽: 도메인 경계 5 voxels = solid
- 내부 격벽: 1mm = 5 voxels = solid
- 채널: fluid
- Z 방향: 전 구간 동일 단면 (Gyroid 아님, 직선 채널)

**검증:** 생성된 마스크의 유체 셀 수 ÷ 전체 내부 셀 수 ≈ 338.7/548 ≈ 0.618

  u_channel = 0.449 m/s, Re = 39.1

### 2.7 PASS 기준

| 항목 | 기준 |
|------|------|

## 3. 추가 검증 지표: S_v (비표면적)

### 3.1 목적

BO 목적함수 f₁인 S_v 산출 로직도 L2에서 검증 가능.

### 3.2 방법

6×6 마스크의 이론 S_v:

```
채널 둘레 = 4 × 3.067mm = 12.27mm (채널당)
총 벽면적 = 36채널 × 12.27mm × L_cat
촉매층 체적 = 23.4mm × 23.4mm × L_cat
S_v_theory = 총 벽면적 / 체적
```

시뮬 쪽 S_v는 voxel 기반으로 산출할 텐데, voxel 표면적은 이론 대비 **10~30% 과대 추정**이 일반적. 이 편차를 정량화해 두면 Gyroid S_v 보정에 사용 가능.

> 이 항목은 PASS/FAIL 대상은 아님. 편차 정량화가 목적.
