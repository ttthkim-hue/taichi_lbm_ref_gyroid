## 0. 현재 진단

### 0.1 dt 고정은 작동하는가?

**예, 작동합니다.** 1k스텝에서 Δρ=0.37%로 LBM 허용 범위(<1%)에 진입 확인. 기존(dt∝1/u_in)에서는 저유속에서도 Δρ=50%였으므로, dt 고정이 문제를 해결한 것이 명확합니다.

다만 1k스텝은 과도 구간(Q_in=8.05, Q_out=0.68)이므로 ΔP 정확도 판정은 수렴 후에만 가능합니다. 수렴까지 ~70분 추가 소요.

### 0.2 남은 불확실성

| 항목 | 상태 | 필요 조치 |
|------|------|-----------|
| L2 저유속 ΔP 정확도 | 수렴 대기 중 | 수렴 후 오차 확인 |
| 주기BC 솔버 | 구현됨, 미검증 | L2 주기BC 실행 |
| Gyroid 이론값 | 없음 | 이론 추산 필요 |

## 2. Phase 2: L2 6×6 다각도 검증 (3가지 방법)

하나의 형상(6×6)에 대해 **3가지 독립적 방법**으로 ΔP를 산출하여 교차 검증합니다. 2개 이상 일치하면 솔버 신뢰도가 확보됩니다.

### 2.1 방법 A — Velocity Inlet + 저유속 (dt 고정)

| 항목 | 값 |
|------|-----|
| 격자 | 131×131×550 |
| u_in | 0.0123 m/s |
| 이론 비교 | HP Fanning 직접 비교 (완전 발달) |
| Δρ 예상 | ~2% |
| 상태 | **실행 중 (수렴 대기)** |

### 2.2 방법 B — 주기BC + 체적력 (원래 유속)

| 항목 | 값 |
|------|-----|
| 격자 | 131×131×550 (Z 주기) |
| ΔP_target | 3.819 Pa (g_lbm으로 부여) |
| 검증 지표 | 정상상태 u_mean vs u_channel_theory(0.449 m/s) |
| Δρ 예상 | ~0% (체적력이므로) |
| 상태 | **미실행** |

**실행 스크립트:** `run_l2_periodic_plan19v.py`

검증 방식이 역방향임에 주의:
- 방법 A: u_in 고정 → ΔP 측정 → 이론 ΔP와 비교
- 방법 B: ΔP 고정(g로 부여) → u_mean 측정 → 이론 u와 비교

**둘 다 같은 물리를 검증하지만 독립적인 경로이므로, 교차 검증 가치가 높습니다.**

### 2.3 방법 C — 투과율(K) 비교

6×6 직선 채널의 투과율은 해석적으로 정확히 산출 가능합니다:

```
단일 정사각 채널(폭 a)의 투과율:
  K_single = a² / (8 × f_correction)
  정사각 단면: K_single = a² × (f·Re)_circ / (f·Re)_square / 8
            ≈ a² × 16 / 56.91 / 8 = a² × 0.03515

a = 3.2 mm (131격자 채널 폭)
K_single = (3.2e-3)² × 0.03515 = 3.60e-7 m²

6×6 매체의 유효 투과율:
  K_eff = K_single × ε  (ε = 공극률 0.6295)
        = 3.60e-7 × 0.6295 = 2.27e-7 m²
```

> ⚠️ 위 수치는 개략 추산. 정사각 채널 투과율 정의와 공극률 보정은 에이전트가 Darcy 법칙으로 재유도할 것.

시뮬에서 K 산출:

```
Darcy: u_superficial = K × ΔP / (μ × L)
→ K_sim = u_in × μ × L / ΔP_sim
```

**방법 A, B, C 모두에서 K를 산출하여 상호 비교.** K가 ±5% 이내 일치하면 솔버 검증 완료.

### 2.4 추가 진단: 채널별 속도 프로파일

정상상태에서 출구 단면(z_out)의 속도 분포를 2D 슬라이스로 저장:

```
vz[x, y, z_out] → numpy → 시각화
```

각 채널 내부에서 **포물선 프로파일**이 형성되어야 합니다. 프로파일이 비대칭이거나 채널 간 차이가 크면 마스크/BC 오류입니다.

> 이건 PASS/FAIL 기준은 아님. 정성적 확인용.

## 3. Phase 3: Gyroid 이론 비교 검증

### 3.1 목적

Gyroid는 HP 같은 해석해가 없으므로, **반경험적 상관식**으로 이론 ΔP를 추산하고 시뮬과 비교합니다. 정밀 일치(5%)가 아니라 **정성적 일치(±30% 이내)와 GHSV 스케일링 관계**를 확인하는 것이 목적입니다.

### 3.2 Gyroid 기준 케이스

| 항목 | 값 |
|------|-----|
| 단위셀 | a = 5 mm |
| 두께 파라미터 | t = 0.3 |
| 예상 공극률 | ε ≈ 0.5 (커널 생성 후 실측) |
| 격자 | 131×131×550 (또는 127), dx=0.2mm |
| 모드 | 주기BC + 체적력 |

### 3.3 이론 ΔP 추산 — Kozeny-Carman 상관식

다공성 매질의 표준 상관식:

```
ΔP/L = 180 × μ × u_s × (1-ε)² / (ε³ × Dp²)

여기서:
  μ = 2.626e-5 Pa·s (200°C 공기)
  u_s = superficial velocity (= u_in)
  ε = 공극률
  Dp = 등가 입자 지름 ≈ 6(1-ε) / S_v
  S_v = 비표면적 (커널 생성 후 실측)
```

> ⚠️ Kozeny-Carman은 입상 충전층 기반이므로 TPMS에 정확하지 않음. 차이가 2배 이상 나올 수 있음.

### 3.4 이론 ΔP 추산 — TPMS 전용 상관식

Femmer et al. (2015) 또는 Al-Ketan et al. (2018)에서 제시한 TPMS 투과율 상관식이 있음:

```
K = C × a² × ε^n

여기서 C, n은 Gyroid 구조 특성 상수 (문헌 참조)
```

> 에이전트는 아래 논문에서 Gyroid sheet 구조의 K-ε 상관식을 검색하여 적용할 것:
> - Femmer et al., Chem. Eng. J., 2015
> - Al-Ketan et al., Adv. Eng. Mater., 2018
> - Peng et al., Chem. Eng. Sci., 2023
> ⚠️ 위 논문이 정확하지 않을 수 있음. 키워드 "Gyroid permeability correlation"로 검색.

### 3.5 다중 GHSV 비교 (스케일링 검증)

동일 Gyroid 형상에서 GHSV만 바꿔서 ΔP 스케일링을 확인합니다:

| GHSV (h⁻¹) | u_in (m/s) | Re 범위 | 이론 ΔP 스케일 |
|-------------|------------|---------|---------------|
| 2,000 | 0.0556 | 매우 낮음 | 1× (기준) |
| 5,000 | 0.1389 | 낮음 | ~6.25× |
| 10,000 | 0.2778 | 중간 | ~25× |

Darcy 영역(Re 충분히 낮으면)에서 ΔP ∝ u이므로 선형 관계가 성립해야 하고, Re가 올라가면 Forchheimer 효과로 ΔP ∝ u²에 가까워집니다.

**주기BC + 체적력 방식이므로, 3개 GHSV 모두 Δρ ≈ 0으로 비압축성 문제 없음.**

실행:

```
Gyroid (a=5, t=0.3) 131×131×550 생성
각 GHSV에 대해 g_lbm을 설정하여 시뮬 실행
정상상태 u_mean 측정
K = u_mean × μ × L / ΔP (각 케이스)
3개 케이스의 K가 ±10% 이내 일치하면 스케일링 검증 PASS
```

이 검증이 통과하면:
- 솔버가 Gyroid 형상을 올바르게 해석하고
- GHSV 변화에 대한 물리적 응답이 정확하며
- BO에서 사용할 수 있다는 것이 확인됩니다.

## 4. 실행 순서 총정리

```
=== Phase 1: L2 저유속 수렴 확인 ===
1. L2 저유속 131격자 수렴 대기 (~70분)
   → PASS/FAIL 판정

=== Phase 2: L2 다각도 검증 ===
2. 방법 B: L2 주기BC 131격자 실행 (~70분)
3. 방법 C: 방법 A, B 결과에서 K 산출 비교
4. 출구 단면 속도 프로파일 시각화 (선택)

=== Phase 3: Gyroid 이론 비교 ===
5. Gyroid (a=5, t=0.3) 커널 생성, ε·S_v 실측
6. Kozeny-Carman + 문헌 상관식으로 이론 ΔP 추산
7. 주기BC로 GHSV 2k/5k/10k 3개 시뮬 (~70분 × 3)
8. K 일정성 + ΔP 스케일링 검증
```

**예상 총 소요: ~6시간 (GPU 연속)**
