## 1. MRT S-matrix 보존량 점검 (최우선)

**목적:** 질량·운동량 보존 보장

**할 일:**
- S 대각 벡터(19개)를 콘솔에 출력하여, 아래 인덱스가 **정확히 0.0**인지 확인

| 인덱스 | moment | 올바른 값 | 0이 아니면 |
|--------|--------|-----------|-----------|
| s₀ | 밀도 (ρ) | 0.0 | 질량 비보존 |
| s₃ | x-운동량 (jx) | 0.0 | 운동량 비보존 |
| s₅ | y-운동량 (jy) | 0.0 | 운동량 비보존 |
| s₇ | z-운동량 (jz) | 0.0 | 운동량 비보존 |

- s₉~s₁₃ (응력 텐서) = `1/τ = 1.681` 인지 확인
- M, inv_M 행렬이 Lallemand & Luo (2000) Table 1과 일치하는지 대조

**판정:** 위 4개가 0.0이고 s₉~s₁₃ = 1/τ이면 PASS. 아니면 수정 후 재확인.

## 2. 스트리밍 + Bounce-back 로직 점검

**목적:** 고체 경계에서 분포함수가 올바르게 반사되는지 확인

**확인 사항:**
- Push 스트리밍 시 **유체 셀만** 이웃으로 전파하는지
- Bounce-back은 **목적지가 고체일 때** 반대 방향으로 반사하는지
- 두 조건이 독립적으로 작동하는지 (고체 셀이 push를 먼저 하고 덮어쓰는 문제 없는지)

**의사코드 (참고용, 코드 그대로 쓰지 말 것):**

```
# 올바른 로직 의도:
for each cell i, each direction s:
    if cell i is fluid:
        ip = i + e[s]
        if ip is fluid:
            F_new[ip, s] = f[i, s]          # 정상 전파
        else:  # ip is solid
            F_new[i, opposite(s)] = f[i, s]  # bounce-back
```

**판정:** 빈 덕트(고체 벽 존재)에서 벽면 근처 속도가 0에 수렴하면 PASS.

## 3. ΔP 측정 위치 수정

**문제:** 현재 Inlet(Z=0), Outlet(Z=NZ-1)에서 측정 중. 이 셀은 BC가 강제된 셀이라 물리적 압력이 아님.

**수정:**
- Inlet 측 측정면: `Z = buf_cells + 5` (BC에서 충분히 안쪽)
- Outlet 측 측정면: `Z = NZ - 1 - buf_cells - 5`
- 각 단면에서 **유체 셀만** 평균 밀도 산출 → `ΔP = (ρ_in − ρ_out) × c_s² × (ρ_phys / ρ_lbm)`

**주의:** 물리 단위 변환 시 `c_s² = 1/3 (격자 단위)`, `ΔP_phys = Δρ_lbm × (dx/dt)² / 3` 임을 확인.

## 4. 수렴 판정 및 로깅 추가

**현재 문제:** 300 스텝은 미수렴 (flow-through ≈ 600 스텝 이상 필요)

**추가할 로깅 항목 (매 1000 스텝):**

| 항목 | 산출 방법 | 정상상태 기준 |
|------|-----------|--------------|
| ΔP | 항목 3의 방식 | 변화율 < 0.1%/1000스텝 |
| 총 질량 Σρ | 전체 유체 셀 밀도 합 | 드리프트 < 0.1% |
| Q_in vs Q_out | 입출구 단면 유량 (Σ uz·유체셀) | 차이 < 1% |
| max(\|u_lbm\|) | 전체 도메인 최대 속도 | < 0.17 (Ma < 0.3) |

**수렴 판정:**
- ΔP 변화율이 3회 연속 0.1% 미만이면 정상상태 도달로 판정
- 최대 스텝 한도 설정 (예: 100,000 스텝), 미수렴 시 발산으로 처리

## 5. 빈 덕트 Level 1 검증 (전체 도메인)

**전제:** 항목 1~4 완료 후 수행

**도메인:** 127 × 127 × 550, dx = 0.2mm (25.4 × 25.4 × 110 mm 빈 덕트)

**이론값 (정사각 단면 Hagen-Poiseuille):**

```
Dh = 25.4 mm (정사각 단면 수력지름 = 변 길이)
Re = u_in × Dh / ν = 0.2778 × 0.0254 / 3.52e-5 = 200.4
f = 56.91 / Re = 0.284
L_cat = 100 mm (측정 구간)
ΔP_theory = f × (L/Dh) × (0.5 × ρ × u²)
           = 0.284 × (0.1/0.0254) × (0.5 × 0.746 × 0.2778²)
           = 0.284 × 3.937 × 0.02878
           ≈ 0.0322 Pa
```

> ⚠️ 위 수치는 수기 계산이므로 반드시 직접 재계산할 것.  
> 정사각 단면 마찰계수 `f·Re = 56.91`은 Incropera 등 교과서 참조.

**PASS 기준:** 시뮬 ΔP vs 이론 ΔP 오차 **< 5%**

## 6. Voxelization 방식 변경

**현재:** STL → Trimesh voxelize → numpy → Taichi Field (CPU, 느림, 누수 위험)  
**변경:** Taichi 커널에서 Gyroid 수식 직접 평가 (GPU, 즉시, 정확)

**Gyroid 수식 (참고용, 코드 그대로 쓰지 말 것):**

```
φ(x,y,z) = sin(2πx/a)·cos(2πy/a) + sin(2πy/a)·cos(2πz/a) + sin(2πz/a)·cos(2πx/a)
solid = 1 if |φ| < t else 0   (sheet 타입)
또는
solid = 1 if φ > t else 0     (네트워크 타입)
```

**커널 내 추가 조건:**
- 덕트 외벽: 도메인 경계 1mm(5 voxels)는 무조건 solid
- 벽 두께 검증: 생성 후 solid 필드에서 최소 벽 두께 ≥ 3 voxels 확인

**STL 출력:** 검증용으로 solid field → numpy → marching_cubes → STL 내보내기 함수 별도 작성

## 7. 격자 독립성 스터디 (Level 3)

**항목 5 PASS 후 수행**

| Level | dx | 격자 크기 | 노드 수 | 메모리 |
|-------|-----|-----------|---------|--------|
| Coarse | 0.4 mm | 64×64×275 | 1.1M | ~0.3 GB |
| Medium | 0.2 mm | 127×127×550 | 8.9M | ~2.2 GB |
| Fine | 0.15 mm | 169×169×733 | 20.9M | ~5.2 GB |

- Gyroid 케이스 1개 (예: a=5mm, t=0) 고정
- 세 격자에서 ΔP, S_v 비교
- Richardson extrapolation → GCI 산출
- GCI < 5% 이면 Medium 격자 채택 확정

## 8. 역류 방지 클리핑 모니터링

**현재:** Outlet BC에서 `uz < 0`이면 0으로 클리핑 → 질량 보존 위반 가능

**추가할 것:**
