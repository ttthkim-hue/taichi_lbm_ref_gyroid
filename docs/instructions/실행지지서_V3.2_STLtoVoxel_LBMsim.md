Taichi LBM 시뮬레이션 검증용으로 사용할 3개의 STL 모델(빈 덕트, 6x6 셀, 자이로이드)을 생성하는 코드를 작성해 줘.

[공통 치수 및 조건]
- 외부 단면: 25.4mm x 25.4mm (1인치 정사각형)
- 외벽 두께: 1.0mm (따라서 내부 빈 공간은 23.4mm x 23.4mm)
- 전체 길이(Z축): 110mm
- 구간 분할: 
  - Inlet Buffer: Z = 0 ~ 5mm (빈 공간)
  - Main Test Section: Z = 5 ~ 105mm (길이 100mm, 여기에만 내부 구조물 배치)
  - Outlet Buffer: Z = 105 ~ 110mm (빈 공간)

[모델별 상세 지시사항]
1. 빈 덕트 (Empty Duct)
- OpenSCAD 코드로 작성.
- 25.4mm 외부 큐브에서 23.4mm 내부 큐브를 빼는 CSG 연산 사용. 길이는 110mm.

2. 레퍼런스 6x6 셀 (Reference 6x6 Cell)
- OpenSCAD 코드로 작성.
- Main Test Section (Z=5~105) 내부에 격자벽 생성.
- 내벽 두께: 1.0mm
- 23.4mm의 내부 공간을 6x6의 채널로 분할. (X, Y 방향으로 각각 5개의 1.0mm 두께의 내벽이 등간격으로 교차되도록 배치할 것)

3. 자이로이드 덕트 (Gyroid Duct)
- OpenSCAD가 암시적 곡면 생성에 취약하므로, 이 모델은 **Python 스크립트**로 작성할 것.
- 라이브러리: `numpy`, `skimage.measure.marching_cubes`, `trimesh` 사용.
- 수식: sin(x)cos(y) + sin(y)cos(z) + sin(z)cos(x) = 0
- Z=5~105 구간의 23.4mm x 23.4mm 내부 공간을 채우는 자이로이드 메쉬를 Marching Cubes로 추출.
- 생성된 자이로이드 메쉬와, 앞서 정의한 110mm 길이의 빈 덕트(외벽 1.0mm) 메쉬를 `trimesh.boolean.union`으로 병합하여 하나의 STL로 Export 할 것.

[출력 요구사항]
1. 빈 덕트용 `.scad` 코드
2. 6x6 셀용 `.scad` 코드
3. 자이로이드 생성 및 병합용 `.py` Python 코드
수식 계산 시 내벽 1mm 두께를 제외한 내부 공간 좌표계가 정확히 일치하도록 신경 써줘.
