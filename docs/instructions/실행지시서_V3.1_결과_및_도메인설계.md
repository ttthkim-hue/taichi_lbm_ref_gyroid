## 2. 이슈별 조치

### 이슈1: 빈 덕트 25.2 → 25.4 mm 통일

- **신규 마스크**: `verify_duct_25.4mm_mask.npy`
- **shape**: 127×127×520 (25.4×25.4×104 mm, 전부 유체)
- ref_dx02와 동일 도메인(520 = 구조 500 + 버퍼 10+10). 빈 덕트 검증 시 이 마스크 사용.

### 이슈2: L_meas 반영

- **솔버**: 시뮬 시작 시 `L_meas_m = (Z_out - Z_in) * dx` [m] 계산 후 `run_info.json`에 기록.
- **파일**: `Results_*/run_info.json` — `Z_in`, `Z_out`, `L_meas_m`, `dx`, `NZ`.
- **Darcy K**: 반드시 `L_meas_m` 사용.  
  `K = u_avg * (ν*ρ) * L_meas_m / dP`  
  (L_cat=0.100 고정 사용 시 오차 발생.)

### 이슈3: Gyroid 외벽 올바른 방식 (방법 A)

- **스크립트**: `scripts/generate_gyroid_wall_wrapper.py`
- **내부**: 23.4×23.4×110 mm → 117×117×550 셀, 동일 TPMS 수식.
- **래퍼**: 외벽 X,Y 5+5셀(1 mm), Z 버퍼 10+10 → **127×127×570**.
- **출력**: `openlb_gyroid/results/gate_a/gyroid_dx02_wall_buf_v31.npy`  
  (기존 gyroid_dx02_wall_buf.npy는 V3 후처리 방식 → v31 사용 권장.)

## 3. 올바른 도메인 설계 (정리)

| 항목 | 값 |
|------|-----|
| 단면 | 127×127 = 25.4×25.4 mm (1 inch) |
| Ref | 127×127×520셀 = 25.4×25.4×104 mm. Z 0~9, 510~519 유체 버퍼, 10~509 구조체. |
| Gyroid (v31) | 127×127×570셀. Z 0~9, 560~569 유체 버퍼; 10~559 구조체. X,Y 0~4, 122~126 고체 외벽(1 mm); 5~121 내부(23.4 mm)에만 gyroid. |
| 빈 덕트 (V3.1) | 127×127×520, 전부 유체. |

**dP 측정**

- Z_in = buf + 2 = 12, Z_out = NZ − 12.
- L_meas = (Z_out − Z_in) × dx [m].  
  예: NZ=520 → L_meas = 496×0.0002 = 0.0992 m; NZ=570 → 546×0.0002 = 0.1092 m.
