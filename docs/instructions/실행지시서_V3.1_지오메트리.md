이슈 1: Empty duct 25.2mm — 일관성 문제
현재:
  Empty duct  → 126×126 = 25.2mm  ← 검증용이라고 별도 정의
  Reference   → 127×127 = 25.4mm (1 inch)
  Gyroid      → 127×127 = 25.4mm (1 inch)

문제:
  빈 덕트만 단면이 달라서 이론식 Dh 계산이 다 달라짐
  Re 계산 → Dh = 25.2mm 기준 vs 25.4mm 기준 → 비교 불가

권장:
  빈 덕트도 127×127 = 25.4mm로 통일
  → 마스크 재생성 필요

이슈 2: 100mm + 버퍼 = 실제 도메인 길이 표기 오류
현재 문서:  Reference / Gyroid 길이 = 100mm

실제 시뮬 도메인:
  ref_dx02:       구조체 500셀 + 버퍼 10+10셀 = 520셀 × 0.2mm = 104mm
  gyroid_dx02:    구조체 550셀 + 버퍼 10+10셀 = 570셀 × 0.2mm = 114mm

dP 측정 위치:
  Z_in  = 12번째 셀   → 물리 위치 2.4mm (구조체 시작 전 2셀)
  Z_out = NZ-12번째  → 물리 위치 L-2.4mm

Darcy K 계산 시 L은 반드시 "측정점 간 거리" 사용
  L_meas = (Z_out - Z_in) × dx  ← L_cat=0.100 그대로 쓰면 오차

이슈 3: 외벽 추가 방식이 근본적으로 잘못됨 ❌
이게 핵심입니다.

  ① 기존 gyroid_dx02.npy = 127×127 (25.4mm, 구조체가 전체 꽉 참)
  ② 여기에 m_wall[0:5] = 1 처리
  ③ 결과: 가장자리 5셀의 gyroid 구조가 그냥 잘려서 벽이 됨

                 ┌──────────────────────┐
  잘못된 결과:   │█ cut  gyroid  cut  █│  25.4mm
                 └──────────────────────┘
                  5셀   117셀    5셀
                  (gyroid가 벽면까지 생성되어 있다가 강제로 잘림)

올바른 방식:
  gyroid를 처음부터 내부 23.4mm (117셀)로 생성하고
  외벽 5셀+5셀을 고체로 래핑해서 전체 127셀 = 25.4mm 구성

                 ┌──────────────────────┐
  올바른 결과:   │█ wall  gyroid  wall █│  25.4mm
                 └──────────────────────┘
                  5셀   117셀    5셀
                  (gyroid가 내부 영역에만 깔끔하게 존재)

따라서 올바른 도메인 설계
전체 도메인:  127 × 127 × (500+20)셀 = 25.4 × 25.4 × 104mm

  X, Y 방향:  셀 0~4    = 고체 외벽 (1mm)
              셀 5~121  = 내부 (23.4mm) ← gyroid/채널이 여기에만 존재
              셀 122~126 = 고체 외벽 (1mm)

  Z 방향:     셀 0~9    = 유체 버퍼 (2mm, inlet)
              셀 10~509 = 구조체 (100mm, L_cat)
              셀 510~519 = 유체 버퍼 (2mm, outlet)

dP 측정:      Z=12 ~ Z=507  → L_meas = 495×0.2mm = 99mm ≈ L_cat ✅

결론: 마스크 생성 파이프라인 수정이 필요
지금처럼 후처리로 외벽을 덧붙이는 방식은 틀렸고, 아래 둘 중 하나를 선택해야 합니다.
방법내용작업량A (권장)generate_gyroid_openlb.py에서 X/Y 도메인을 23.4mm로 줄이고, 래퍼 스크립트에서 외벽 추가 후 127셀로 패딩중간BOpenLB가 이미 외벽 포함 25.4mm로 생성하고 있다면 현재 maske를 그대로 사용 (후처리 외벽 추가 제거)낮음
방법 B가 가능한지 확인하는 것이 먼저입니다:
bash# OpenLB가 생성한 원본 마스크에서
# 가장자리가 이미 고체인지 확인
python3 << 'EOF'
import numpy as np
m = np.load('/mnt/h/openlb_gyroid/results/gate_a/gyroid_dx02.npy')
NX, NY, NZ = m.shape
print(f"X=0  면 solid비율: {m[0,:,NZ//2].mean():.3f}")   # 1.0이면 이미 외벽 있음
print(f"X=-1 면 solid비율: {m[-1,:,NZ//2].mean():.3f}")
print(f"Y=0  면 solid비율: {m[:,0,NZ//2].mean():.3f}")

# → 모두 1.0이면 OpenLB가 이미 외벽 포함해서 생성한 것 → 후처리 불필요
# → 0에 가까우면 외벽 없이 생성한 것 → 방법 A 필요
