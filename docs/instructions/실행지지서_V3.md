## ⚡ 에이전트 트리거 (최우선 실행)

```
이 파일을 열면 아래 순서를 자동 시작:

  [사전 검증 — 시뮬 전]
  STEP M-1  마스크 3종 현황 진단 (solid_frac, 반전 여부, 외벽 유무)
  STEP M-2  Empty duct 마스크 수정 (반전 또는 재생성)
  STEP M-3  Gyroid 외벽 1mm 추가
  STEP M-4  버퍼 10셀 추가 (ref / gyroid)
  STEP M-5  마스크 시각화 확인 (슬라이스 출력)

  [솔버 수정]
  STEP 0-A  apply_bc → Zou-He velocity inlet + pressure outlet
  STEP 0-B  dP 측정 → 유동장 실측 기반
  STEP 0-C  u_latt 단위 전달 확인
  STEP 0-D  설정 검증 출력
  STEP 0-E  질량 모니터링 (유체 셀 기반)

  [시뮬 실행]
  STEP 1    빈 덕트 검증 → Level 1 PASS 필수
  STEP 3    Reference dx02/dx04
  STEP 4    Gyroid cross-check
  STEP 5    BO 파이프라인 연결
  STEP 6    BO Stage 1

  STEP 1 FAIL → 즉시 중단 후 보고
```

## 확정 실패 원인 (V1~V2 분석 종합)

```
[원인 1] Empty duct 마스크가 전부 solid (반전)  ← 이미지 확인으로 신규 발견
  → 중간 도메인 전체 bounce-back → 유동장 없음
  → Uin=0.74 m/s, 질량 불균형 48.6%의 진짜 원인

[원인 2] Gyroid 외벽 1mm 없음  ← 이미지 확인으로 신규 발견
  → 유체가 측면으로 누설(bypass) → K 과소 측정
  → SCR 실제 구조와 불일치 → BO 결과 신뢰성 저하

[원인 3] BC 타입 불일치
  → apply_bc 내부 u=0 고정, u_in_target 미사용
  → dP = BC 임의 Δρ 기반 (유동장 실측 아님)

[원인 4] 버퍼 없음 (ref_dx02/ref_dx04)
  → BC면 = 구조체 시작면 → 입출구 효과 + BC 충돌
```

## STEP M-3: Gyroid 외벽 1mm 추가

### M-3-1 외벽 필요 이유

```
외벽 없을 때:
  유체가 gyroid 측면(X=0, X=NX-1, Y=0, Y=NY-1)으로 통과 가능
  → 실제 채널 우회(bypass) → K 과대 측정
  → SCR 촉매 외벽(canning) 구조와 불일치

외벽 추가 후:
  X=0~4, X=NX-5~NX-1  (5셀 = 1mm @ dx=0.2mm)  → 고체(1)
  Y=0~4, Y=NY-5~NY-1  → 고체(1)
  Z는 건드리지 않음 (inlet/outlet BC면)
```

### M-3-2 외벽 추가 스크립트

## STEP M-4: 버퍼 10셀 추가 (ref / gyroid)

```python
import numpy as np

def add_buffer(mask, buf=10):
    """
    Z 방향 앞뒤에 유체(0) 버퍼 추가.
    mask = (NX, NY, NZ_cat),  반환 = (NX, NY, NZ_cat + 2*buf)
    """
    NX, NY, NZ_cat = mask.shape
    out = np.ones((NX, NY, NZ_cat + 2*buf), dtype=mask.dtype)
    out[:, :, buf:buf+NZ_cat] = mask
    out[:, :, :buf]           = 0   # inlet 버퍼 = 유체
    out[:, :, buf+NZ_cat:]    = 0   # outlet 버퍼 = 유체
    print(f"  버퍼 추가: NZ {NZ_cat} → {NZ_cat+2*buf}  (buf={buf}셀 양쪽)")
    return out

BUF = 10

# ref_dx02 (외벽 없음, 버퍼만)
```

라인 296–340 구간을 아래 코드로 전체 교체:

```python
@ti.kernel
def apply_bc(u_in_target: ti.f64, current_step: ti.i32):
    """
    Inlet  (Z=0)    : Zou-He 속도 고정 inlet
                      u = (0, 0, u_lb),  ρ는 내부 유동에서 float
    Outlet (Z=NZ-1) : 압력(밀도) 고정  ρ_out = 1.0 (lattice)
    """
    ramp = ti.min(1.0, ti.cast(current_step, ti.f64) / 5000.0)
    u_lb = u_in_target * ramp   # lattice 단위, RAMP 적용

    # ── INLET (Z=0) ──────────────────────────────────────────────────
    for i, j in ti.ndrange(NX, NY):
        if mask[i, j, 0] == 0:
            rho_near = 0.0
            for q in ti.static(range(19)):
                rho_near += f[i, j, 1][q]
            rho_near = ti.max(rho_near, 0.5)
            for q in ti.static(range(19)):
                f[i, j, 0][q] = feq_q(rho_near, 0.0, 0.0, u_lb, q)

    # ── OUTLET (Z=NZ-1) ──────────────────────────────────────────────
    for i, j in ti.ndrange(NX, NY):
        if mask[i, j, NZ-1] == 0:
            rho_tmp = 0.0
            ux_tmp = uy_tmp = uz_tmp = 0.0
            for q in ti.static(range(19)):
                rho_tmp += f[i, j, NZ-2][q]
                ux_tmp  += f[i, j, NZ-2][q] * ex[q]
                uy_tmp  += f[i, j, NZ-2][q] * ey[q]
                uz_tmp  += f[i, j, NZ-2][q] * ez[q]
            rho_tmp = ti.max(rho_tmp, 0.5)
            ux_out  = ux_tmp / rho_tmp
            uy_out  = uy_tmp / rho_tmp
            uz_out  = uz_tmp / rho_tmp
            for q in ti.static(range(19)):
                f[i, j, NZ-1][q] = feq_q(1.0, ux_out, uy_out, uz_out, q)
```

```python
def calc_K(out_dir, L=0.1, nu=3.52e-5, rho=0.746):
    df   = pd.read_csv(f'{out_dir}/simulation_log.csv')
    last = df.tail(200).mean(numeric_only=True)
    u    = last.get('u_avg_fluid', last.get('Uin', 0))
    dP   = last.get('dP_pascal', 0)
    return u * (nu*rho) * L / dP if dP > 0 else 0
```
